Hvorfor samles dette inn

denne informasjonen samles inn for og kunne oprette konto, lagre brukerrelaterte resultater og gi mulighet for at brukeren kan slette kontoen sin.

lagring og sikkerhet

passord lagres kun som hash + salt.

brukere lagres kun midlertidig uten database.

dine rettigheter
du kan be om innsyn og du kan slette kontoen din (da vil persondata fjernes).

samtykke
du må aktivt samtykke til vilkårene for å oprette konto, og du kan trekke samtykke ved å slette kontoen.
